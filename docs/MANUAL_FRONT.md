# คู่มือการใช้งานหน้าบ้าน

คู่มือนี้สำหรับผู้ใช้งานทั่วไปที่ต้องการแจ้งปัญหา ติดตามเคส ส่งข้อมูลเพิ่มเติม และติดต่อทีมงานผ่านระบบ HealthHelp

## หน้าที่ใช้งาน
- `/`
- `/track`
- `/contact`

## Task 1: แจ้งปัญหาใหม่จากหน้าแรก
- [ ] เปิดหน้า `/`
- [ ] กรอกข้อมูลผู้แจ้ง เช่น ชื่อ เบอร์โทร อีเมล หรือ Line ID
- [ ] เลือกหมวดหมู่ปัญหา
- [ ] กรอกหัวข้อปัญหาและรายละเอียด
- [ ] แนบไฟล์หรือรูปภาพประกอบ หากมี
- [ ] ตรวจสอบข้อมูลให้ครบถ้วน
- [ ] กดส่งแบบฟอร์มแจ้งปัญหา
- [ ] บันทึก `เลขที่เคส` และ `รหัสติดตาม` ที่ระบบแสดงหลังส่งสำเร็จ

## Task 2: ติดตามสถานะเคสด้วยรหัสติดตาม
- [ ] เปิดหน้า `/track`
- [ ] เลือกแท็บ `รหัสติดตาม`
- [ ] กรอกรหัสติดตามของเคส
- [ ] กดปุ่ม `ค้นหาเคส`
- [ ] ตรวจสอบข้อมูลเคส เช่น สถานะ ความเร่งด่วน วันที่แจ้ง และผู้รับผิดชอบ
- [ ] อ่านไทม์ไลน์เพื่อดูความคืบหน้าของเคส

## Task 3: ค้นหาประวัติเคสย้อนหลังด้วยเบอร์โทรศัพท์
- [ ] เปิดหน้า `/track`
- [ ] เลือกแท็บ `เบอร์โทรศัพท์`
- [ ] กรอกเบอร์โทรของผู้แจ้ง
- [ ] กดปุ่ม `ค้นหาเคส`
- [ ] ตรวจสอบรายการเคสย้อนหลังทั้งหมด
- [ ] คลิกเคสที่ต้องการเพื่อเปิดดูรายละเอียด

## Task 4: ส่งข้อความตอบกลับเจ้าหน้าที่จากหน้าติดตามเคส
- [ ] เปิดรายละเอียดเคสในหน้า `/track`
- [ ] เลื่อนลงมาที่ส่วน `ตอบกลับ / ให้ข้อมูลเพิ่มเติม`
- [ ] พิมพ์ข้อความที่ต้องการแจ้งเพิ่มเติม
- [ ] แนบรูปหรือไฟล์ หากต้องการ
- [ ] ตรวจสอบ preview ไฟล์ก่อนส่ง
- [ ] กดปุ่ม `ส่ง`
- [ ] ตรวจสอบว่าข้อความและไฟล์ปรากฏในไทม์ไลน์การสนทนา

## Task 5: ดูไฟล์แนบและรูปภาพในไทม์ไลน์
- [ ] เปิดหน้ารายละเอียดเคส
- [ ] เลื่อนไปที่ไทม์ไลน์การสนทนา
- [ ] คลิกรูปภาพเพื่อขยายดูแบบเต็มหน้าจอ
- [ ] คลิกลิงก์ไฟล์เพื่อเปิดหรือดาวน์โหลดเอกสารแนบ

## Task 6: ให้คะแนนความพึงพอใจหลังปิดเคส
- [ ] เปิดหน้า `/track` และค้นหาเคสที่สถานะ `แก้ไขแล้ว` หรือ `ปิดเคส`
- [ ] เลือกคะแนนความพึงพอใจ 1-5 ดาว
- [ ] กรอกความคิดเห็นเพิ่มเติม หากมี
- [ ] กดปุ่ม `ส่งคะแนน`

## Task 7: ติดต่อทีมงานจากหน้าติดต่อเรา
- [ ] เปิดหน้า `/contact`
- [ ] กรอกชื่อ-นามสกุล
- [ ] กรอกเบอร์โทรศัพท์
- [ ] กรอกอีเมล หากมี
- [ ] กรอกหัวข้อและรายละเอียดข้อความ
- [ ] กดปุ่ม `ส่งข้อความ`

## ข้อควรทราบ
- [ ] ผู้ใช้งานควรเก็บ `รหัสติดตาม` ไว้ทุกครั้งหลังแจ้งเคส
- [ ] หากค้นหาเคสไม่พบ ให้ตรวจสอบการพิมพ์รหัสติดตามหรือเบอร์โทรอีกครั้ง
- [ ] ไฟล์แนบที่ส่งจากผู้ใช้งานจะถูกเก็บในระบบและ sync ไปยังชีต `Attachments`
