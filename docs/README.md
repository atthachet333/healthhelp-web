# คู่มือการใช้งาน HealthHelp

เอกสารชุดนี้จัดทำเพื่อให้เปิดอ่านได้ง่ายทั้งบน Cursor และ GitHub โดยแยกตามกลุ่มผู้ใช้งาน

## เอกสารหลัก
- [คู่มือหน้าบ้าน](./MANUAL_FRONT.md)
- [คู่มือหลังบ้าน](./MANUAL_BACKOFFICE.md)

## เหมาะสำหรับใคร
- `MANUAL_FRONT.md`: สำหรับผู้แจ้งปัญหา ผู้ใช้งานทั่วไป หรือทีมที่ต้องการสอนการใช้งานฝั่งหน้าเว็บ
- `MANUAL_BACKOFFICE.md`: สำหรับเจ้าหน้าที่ ผู้ดูแลระบบ หัวหน้างาน และทีมงานที่ดูแลเคสในระบบ

## วิธีใช้งานเอกสาร
- เริ่มจากอ่าน `README.md` ที่ root ของโปรเจกต์ เพื่อดูภาพรวมระบบ
- หากต้องการคู่มือแบบ step-by-step ให้เปิดเอกสารตามหน้าที่ใช้งาน
- หากต้องการพิมพ์หรือส่งต่อทีมงาน แนะนำใช้ไฟล์ในโฟลเดอร์ `docs/` โดยตรง
